Sistema de Inventário Simples do IFPE

Requisitos

- Python 3.10
- Tkcalendar
- Pillow

Passo a passo para instalação e execução:

* Descompactar a pasta no computador

# unzip exemplo.zip

* Criar venv python 3.10 no diretório descompactado

$ cd exemplo
$ python -m venv .venv

* Instalar dependencias

$ pip install -r requirements.txt

Rodar chamando main.py

$ python main.py
